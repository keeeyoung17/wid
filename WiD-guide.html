시작 전 — 데이터 저장 방식 이해

WiD는 데이터를 브라우저의 localStorage에 저장해. 폴더나 파일로 눈에 보이지 않고, 브라우저 내부에 자동으로 저장되는 방식이야.

💡 WiD는 URL 기준으로 데이터를 저장해. 항상 같은 URL로 접속하면 데이터가 그대로 유지돼. URL이 달라지면 다른 저장 공간을 사용하게 되니 주의해.
1
항상 같은 URL로 접속
전달받은 링크 (예: username.github.io/wid)를 항상 같은 브라우저에서 열어. 크롬에서 저장한 데이터는 사파리에서 보이지 않아.
2
즐겨찾기 등록 추천
URL을 브라우저 즐겨찾기(북마크)에 등록해두면 매번 찾을 필요 없이 바로 접속 가능. 아이폰은 홈 화면에 추가하면 앱처럼 실행돼.
3
정기 백업 권장
브라우저 초기화 또는 기기 교체 시 데이터가 사라질 수 있어. 설정 탭 → 데이터 내보내기로 주기적으로 JSON 백업해둬.
⚠ 기기를 바꾸거나 브라우저를 초기화하면 데이터가 사라져. 새 기기에서 쓰고 싶으면 기존 기기에서 설정 → 데이터 내보내기 후 새 기기에서 데이터 가져오기로 이전할 수 있어.
01

앱 설치 — 아이폰 홈 화면에 추가

WiD는 아이폰 사파리에서 홈 화면에 추가하면 진짜 앱처럼 실행돼. 앱스토어 설치 없이 아이콘이 생기고 전체화면으로 열려.

1
사파리로 링크 접속
반드시 사파리(Safari)로 열어야 해. 크롬이나 다른 브라우저는 홈 화면 추가가 안 돼.
2
공유 버튼 탭
화면 하단 가운데 □↑ (공유 버튼) 탭. 아이폰 기종에 따라 하단 또는 상단 주소창 옆에 있어.
3
홈 화면에 추가
메뉴에서 "홈 화면에 추가" 찾아서 탭. 스크롤 내려야 보일 수 있어.
✓
완료
홈 화면에 WiD 아이콘이 생겨. 이제 앱처럼 탭해서 실행하면 돼.
⚠ 홈 화면 추가 후에는 항상 홈 화면 WiD 아이콘으로 열어야 해. 사파리로 다시 들어가면 다른 저장 공간을 사용할 수 있어.
02

API 키 발급

WiD의 AI 기능(브리핑, 업무 분석, 회고)을 쓰려면 API 키가 필요해. Gemini는 무료, Claude는 소액 유료야.

Gemini API
무료 · 추천
  • 구글 계정만 있으면 OK
  • 무료 티어로 충분히 사용 가능
  • aistudio.google.com 에서 발급
Claude API
유료 · 고성능
  • 더 정확한 AI 분석
  • 월 $0.20 수준 (매우 저렴)
  • console.anthropic.com 에서 발급

Gemini API 키 발급 (무료) — 상세 안내

처음 하는 거라 헷갈릴 수 있으니 스크린샷 없이도 따라할 수 있게 단계별로 설명할게.

1
Google AI Studio 접속
aistudio.google.com 접속.
구글 계정으로 로그인 (Gmail 계정 있으면 바로 가능).
2
Get API Key 클릭
왼쪽 사이드바에서 "Get API key" 클릭.
없으면 화면 상단 또는 좌측 메뉴를 스크롤해봐.
3
새 프로젝트로 생성 (중요!)
"Create API key in new project" 클릭.
⚠ 기존 구글 클라우드 프로젝트 선택 말고 반드시 새 프로젝트로 만들어야 무료 티어가 적용돼.
기존 프로젝트로 만들면 유료 청구될 수 있어.
4
API 키 복사
생성된 키가 화면에 표시돼. AIza로 시작하는 긴 문자열이야.
복사 버튼 클릭해서 복사해둬. 이 키는 다시 볼 수 없으니 어딘가 메모해두는 게 좋아.
5
WiD 설정 탭에 입력
WiD 열고 → 설정 탭 → AI 제공자 "Gemini" 선택
→ API 키 붙여넣기 → 저장 → 연결 테스트 클릭
→ "✓ 연결됨" 뜨면 성공!
❗ 연결 테스트 실패 시: 새 프로젝트로 다시 만들어봐. 기존 구글 클라우드 프로젝트에 연결된 키는 무료 티어 한도가 0일 수 있어. "Create API key in new project" 를 반드시 선택해야 해.

Claude API 키 발급 (유료)

1
Anthropic 콘솔 접속
console.anthropic.com → 회원가입 또는 로그인
2
결제 수단 등록
Settings → Billing에서 카드 등록. 실제 사용량만 청구 (보통 월 $0.20 미만)
3
API 키 발급 및 입력
API Keys → Create Key → sk-ant-로 시작하는 키 복사 → WiD 설정 탭에 입력

대시보드 탭

출근하면 가장 먼저 보는 화면. 오늘 뭘 해야 하는지 AI가 정리해줘.

🚨
얼럿 스트립
기한 초과(D+N) · D-3 이내 긴급 · D-7 이내 주의 순으로 상단에 자동 표시. 클릭하면 해당 업무 상세로 이동.
📊
상단 4개 스탯 카드
이번달 달성률 / 오늘 리소스 (8칸 블록, 초과 시 빨간색) / 얼럿 현황 / 이번주 회고 요약. 회고 카드 클릭하면 회고 탭으로 이동.
✦
AI 브리핑
오늘 날짜 기준 전체 업무 분석 → 우선순위 + 업무별 전략 제안. 하루 한 번 자동 생성, 새로고침으로 재생성 가능. 완료/삭제/대기 중 업무는 자동 제외. 쓸수록 내 패턴을 학습해서 브리핑이 정교해져.
✅
오늘 할 일
AI 우선순위 순 정렬. 업무명 + 오늘 해당 하위 일정 + AI 전략 한 줄 표시. 카드 클릭하면 팝업으로 전체 하위 일정 확인 및 체크 가능.
🔁
오늘 루틴
오늘 날짜/요일 기준으로 해당하는 루틴 자동 표시. 체크하면 오늘 완료 처리, 다음날 자동 리셋.
📊
이번주 스케줄
월~금 범위 업무 Gantt 바. 마감일 ◆ 마커 (초과 빨강 · D-3 주황 · D-7 이후 하늘). AI 브리핑과 나란히 표시.
05

업무 탭

업무 상태 종류

진행중 대기 중 예정 완료됨 삭제됨

대기 중은 "고객사 컨펌 대기", "자료 수령 대기" 같이 내가 지금 당장 할 수 없는 업무야. 대기 중 업무는 오늘 할 일과 AI 브리핑에서 자동 제외돼. 완료됨/삭제됨 업무는 전체 필터에서 안 보이고 각각의 필터에서만 확인 가능.

빠른 캡처 (Quick Add)

모든 탭 우측 하단에 ＋ 플로팅 버튼이 있어. 회의 중에, 통화 중에 급하게 생각난 업무를 빠르게 저장할 때 써.

⚡
자연어 날짜 입력 지원
"내일", "다음주 금요일", "7/10", "이번주 목" 등 자연어로 입력하면 자동으로 날짜로 변환돼.
📌
미완성 배지
빠른 캡처로 저장된 업무는 "미완성" 배지로 표시. 나중에 상세 정보 추가 가능.

새 업무 추가 — 2단계 폼

상단 ＋ 새 업무 버튼 클릭 시 상세 입력 가능:

1
기본 정보 입력
업무 구분 · 세부 업무 · 업무명 · 시작일 · 마감일 · 우선도 · 예상 소요시간 · 업무 설명 · 자료 링크/첨부. 업무 구분과 세부 업무는 직접 추가/삭제 가능. 날짜는 자연어로 입력 가능.
✦
AI 분석
설명 기반으로 소요시간 조정 + 하위 일정 자동 생성. 주말 자동 제외 (시작일/마감일에 주말 포함 시에만 주말 일정 생성).
2
일정 확인 및 저장
AI 생성 하위 일정 확인. 이름·날짜·시간 직접 수정, ⠿ 핸들 드래그로 순서 변경, ＋ 추가 · ✕ 삭제 가능.

상세 드로어

카드 클릭 시 우측에 뜨는 상세 창:

✏️
인라인 직접 수정
우선도 칩 클릭으로 즉시 변경. 하위 일정 이름·날짜·시간 클릭해서 바로 편집. 메모 영역 클릭해서 바로 입력.
🔄
수정 모달 + AI 재분석
드로어 상단 "수정" 버튼 → 기존 데이터 채워진 모달 오픈. "AI 재분석" 버튼으로 하위 일정 새로 생성 가능.
⏱
완료 시 실제 소요시간 기록
완료 처리 시 "실제로 얼마나 걸렸어?" 입력창 표시. 입력하면 회고 탭의 시간 추정 정확도 분석에 반영됨. 스킵 가능.
🗑
삭제 및 복구
삭제해도 완전히 지워지지 않고 "삭제됨" 필터에서 확인 가능. 복구 버튼으로 되살리기 가능.
06

루틴 탭

매일/매주/매월 반복되는 업무를 등록해두면 대시보드 "오늘 루틴" 에 자동으로 표시돼.

🌅
일간 루틴
매일 또는 특정 요일(월/화/수/목/금/토/일 복수 선택) 반복.
📅
주간 루틴
몇 주마다 + 요일 선택. 예: 격주 화요일.
🗓
월간 루틴
매월 특정 일자. 예: 매월 1일, 매월 마지막 날.
⏸
활성/비활성 토글
비활성으로 설정하면 대시보드에 표시 안 됨. 삭제하지 않고 잠시 꺼둘 수 있어.
07

캘린더 탭

월간 Gantt 타임라인으로 전체 업무 일정을 한눈에 파악.

📊
월간 Gantt 타임라인
하위 일정 시작일 기준으로 바 시작, 마감일에 끝남. 업무 구분별 색상 구분. 주말 컬럼 연하게 처리. 진행중/예정/완료 구분 색상.
◆
마감일 마커
기한 초과 빨강 · D-3 이내 빨강 · D-7 이내 주황 · 이후 하늘 · 완료 민트.
🌫
다음달 미리보기
이번달 마지막 날 이후 14일치 흐리게 표시. 다음달로 이어지는 업무도 흐리게 연장 표시.
👆
바 클릭 → 팝업
Gantt 바 클릭하면 해당 업무 상세 팝업 오픈. 하위 일정 체크 및 업무 탭으로 이동 가능.
💡 완료 처리된 업무는 캘린더에서 자동으로 사라져. 복구하면 다시 표시돼.
08

회고 탭

업무를 완료할수록 데이터가 쌓이고 AI가 내 패턴을 분석해줘. 쓸수록 브리핑이 정교해지는 이유야.

📅
기간 선택
이번주 / 지난주 / 이번달 / 전체 중 선택해서 조회.
📈
핵심 지표
완료 업무 수 + 스트릭 🔥 · 시간 추정 정확도 % · 가장 많이 한 세부 업무 · 총 실제 투입 시간.
⏱
시간 추정 정확도
세부 업무별 예상 vs 실제 시간 바 차트. 초과 빨간색 · 여유 민트색. "보도자료: 예상 3h → 실제 4.2h (40% 초과)" 형식.
📆
요일별 생산성 패턴
월~금 요일별 완료율 차트. 가장 생산적인 요일 강조 표시.
📖
Logbook
날짜별 완료 업무 목록. 업무명 · 예상 시간 · 실제 시간 · 업무 구분.
✦
AI 심층 분석
"AI 분석 요청" 버튼 클릭 시 누적 패턴 전체 분석. 반복 시간 초과 패턴 · 생산성 높은 요일 · 마감 관리 패턴 · 다음 기간 개선 제안 3가지 제공. 데이터 5개 이상 쌓이면 활성화.
🗓
주간 계획 모드
회고 탭 상단 토글로 이번주 계획 뷰 전환. 월~금 컬럼에 업무 드래그로 배치, 요일별 시간 합산 표시. 매주 월요일 대시보드에 "이번주 계획 세우기" 배너 표시.
✓ 업무 완료 시 실제 소요시간을 입력할수록 분석이 정확해져. 스킵해도 되지만 입력할수록 AI 브리핑 품질이 올라가.

설정 탭
